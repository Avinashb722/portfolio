# Images Directory

This directory contains media files for the project:

- `robot_demo.gif` - Animated demo of robot in action
- `robot_front.jpg` - Front view of the assembled robot
- `robot_circuit.jpg` - Circuit board and wiring
- `circuit_diagram.png` - Schematic diagram
- `line_following_logic.png` - Algorithm flowchart

## Image Guidelines

- Use high-quality images (min 800px width)
- Keep file sizes under 5MB
- Use descriptive filenames
